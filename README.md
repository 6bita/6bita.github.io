# 6bita.github.io
Publicaciones de El 6Ceibita

Los posteos o ingresos del blog se encuentran en el directorio `_posts`. Para escribir en el blog simplemente ingresar un documento formato md con el correspondiente front matter en este directorio `_posts` del root directorio. En el front matter aparece la categoría o el tema al cual pertenece el documento asi tambien como el nombre del autor del documento.
